# Requirements

To debug and run this Rust project, you need:

1.  **Rust Toolchain**: Install via [rustup.rs](https://rustup.rs/).
2.  **VS Code Extensions**:
    *   `rust-lang.rust-analyzer` (Rust Analyzer)
    *   `vadimcn.vscode-lldb` (CodeLLDB) for debugging.
3.  **Dependencies**: Managed via `Cargo.toml`. Run `cargo build` to install.
