What this software does

  • Renders millions of geometries fast (instanced draw calls + automatic level-of-detail).
  • Imports geometry from a text DSL, Excel/CSV/JSON tables and STL/OBJ/glTF models.
  • Visualizes ML datasets (NPY/NPZ/CSV/Parquet/IDX) as labelled 3D point clouds with PCA.
  • Full scene editor: 3D picking, multi-selection, undo/redo, CAD-like camera, measurement tool.

Requires macOS 10.13 or later. Universal build (Apple Silicon & Intel).
