Qué hace este software

  • Renderiza millones de geometrías rápidamente (llamadas de dibujo instanciadas + nivel de detalle automático).
  • Importa geometría desde un DSL de texto, tablas Excel/CSV/JSON y modelos STL/OBJ/glTF.
  • Visualiza datos de ML (NPY/NPZ/CSV/Parquet/IDX) como nubes de puntos 3D etiquetadas con PCA.
  • Editor de escena completo: selección 3D, multiselección, deshacer/rehacer, cámara CAD, herramienta de medición.

Requiere macOS 10.13 o posterior. Compilación universal (Apple Silicon e Intel).
