Cosa fa questo software

  • Renderizza milioni di geometrie velocemente (draw call istanziate + livello di dettaglio automatico).
  • Importa geometrie da un DSL testuale, tabelle Excel/CSV/JSON e modelli STL/OBJ/glTF.
  • Visualizza dataset di ML (NPY/NPZ/CSV/Parquet/IDX) come nuvole di punti 3D etichettate con PCA.
  • Editor di scena completo: selezione 3D, multi-selezione, annulla/ripeti, telecamera CAD, strumento di misura.

Richiede macOS 10.13 o successivo. Build universale (Apple Silicon e Intel).
