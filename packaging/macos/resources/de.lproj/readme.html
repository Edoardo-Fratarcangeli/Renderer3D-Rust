Was diese Software macht

  • Rendert Millionen von Geometrien schnell (instanzierte Draw-Calls + automatischer Detailgrad).
  • Importiert Geometrie aus einer Text-DSL, Excel/CSV/JSON-Tabellen und STL/OBJ/glTF-Modellen.
  • Visualisiert ML-Datensätze (NPY/NPZ/CSV/Parquet/IDX) als beschriftete 3D-Punktwolken mit PCA.
  • Vollständiger Szeneneditor: 3D-Auswahl, Mehrfachauswahl, Rückgängig/Wiederholen, CAD-Kamera, Messwerkzeug.

Erfordert macOS 10.13 oder neuer. Universal-Build (Apple Silicon und Intel).
